FloydRoseTremoloTuning
======================

Premier programme python : 

	- permet de saisir le signal du micro à l'aide du module PyAudio
	- analyse ce signal pour en détecter la fréquence fondamentale (pour l'instant une seule méthode utilisée, 
par la suite une deuxième au moins sera implémentée pour pouvoir les comparer dans le cas précis d'une note de guitare)
	- renvoie la note jouée si la corde est "accordée" à une erreur de 0,1Hz près, "Corde fausse" dans le cas contraire.
